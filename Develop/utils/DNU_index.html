#Weather-Forecast

##Description:

  • This application was built to help users look up their local weather at the current time, and also into the future

##Table of Content:

  • Description
  • Installation Instructions

##Installation Instructions:

  • The base code should be downloaded from the GitHub repository. Any additional features needed are also available on our GitHub

##Usage:

  • A user will enter the location they want to review the weather for. They will enter this information into the search bar, and the application will return the weather for that city.
